Vamos aprender sobre Operadores em JavaScript.

JavaScript possui vários operadores, entre eles estão:

Aritméticos

Atribuição

Relacionais

Lógicos

Ternários

Vamos começar pelos Aritméticos.

+ = Adição

- = Subtração

* = Multiplicação

/ = Divisão

% = Divisão inteira

** = Exponenciação

Temos sempre de atentar à precedência dos operadores.

A precedência é:

1 - Tudo o que tiver entre parênteses

2 - Exponenciação

3 - Multiplicação / Divisão

4 - Adição e subtração

5 - Se operadores são iguais, a conta é feita da esquerda para a direita.

Operador de Atribuição

= Sinal de igual é usado para atribuição de valores à variáveis.

Operadores Relacionais

> - Maior que

< - Menor que

>= - Maior ou igual que

<= - Menor ou igual que

== - Igual a

!= - Diferente de

Operador de Identidade ou Igualdade Restrita

É o operador que testa se ambos os objetos possuem mesmo valor e tipo.

=== - Idêntico

!== - Não idêntico, ou, desigual restrito

Operadores Lógicos

&& - Conjunção ou Operador E

|| - Disjunção ou Operador OU

! - Operador Negação

Operadores Ternários

?

:

teste ? true : false

media >= 7 ? "Aprovado" : "Reprovado"
