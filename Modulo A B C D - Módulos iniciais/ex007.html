Aprendendo a Manipular o DOM - Document Object Model

Vamos aprender a fazer a Árvore DOM do meu site.

A janela do navegador é um objetio chamado window...

Temos o objeto location, o documento, o history, e muitos outros objetos.

Dentro do documento temos o objeto HTML e seus filhos, head e body.

Vamos aprender 5 métodos de acesso aos elementos da árvore DOM, e podemos selecionar estes elementos pra poder navegar.

Podemos acessar:

Por marca

Por ID

Por Nome

Por Classe

Por Seletor

Por Marca - getElementsByTagName()

Estas são as maneiras de selecionarmos elementos do documento html utilizando o código javascript, isto possibilita a integração destes elementos e a manipulação através do código Js.
